git branch <branch_name> - створення гілки
git branch - відображення списку локальних гілок. Символ * позначатиме поточну гілку
git branch -v - відображення списку локальних гілок з останньою фіксацією
git branch --no-merged - відображає локальні гілки, які ще не були злиті в поточну гілку (мають відмінності)
git branch --merged - відображає локальні гілки, які ще вже були злиті в поточну гілку (не мають відмінностей). Можна використовувати як список кандидатів на вилучення.
git branch -d - вилучення логальної гілки
git branch -D - примусове вилучення логальної гілки (гілки що не буди злиті не будуть вилучені при застосуванні параметру -d)
